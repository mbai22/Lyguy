# Lyguy — Pressbook officiel

Site vitrine de l'artiste **Lyguy**, rappeur tchadien. Présentation de son univers, discographie, agenda, galerie, actualités et boutique.

## Stack

- HTML5 statique (multi-pages via show/hide)
- CSS3 (variables, flexbox, grid, scroll-snap carousels, responsive 380px+)
- JavaScript vanilla (navigation SPA, carrousels, animations scroll, formulaire de contact)
- Phosphor Icons

## Pages

1. **Accueil** — hero, bio, univers, actus, boutique, booking
2. **Bio** — identité, frise chronologique (1998–2023)
3. **Univers** — inspirations, collaborations, positions
4. **Discographie** — albums TCHAD, ABA
5. **Agenda** — concerts passés et à venir
6. **Actualité** — articles et entrevues
7. **Galerie** — 12 photos
8. **Boutique** — albums, packs, merch

## Développement

```bash
# Servir localement
npx serve .
```

Aucun build nécessaire — ouvrir `index.html` dans un navigateur suffit.
